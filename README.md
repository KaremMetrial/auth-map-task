## Authentication + Interactive Map

### Features
- User registration & login
- Protected map page
- Browser geolocation
- Interactive map using Leaflet
- Nearby hotels markers (dummy data)

### Tech Stack
- Laravel
- Blade
- MySQL
- Leaflet.js
- HTML / CSS / JS

### Setup
1. Clone repository
2. Run `composer install`
3. Configure `.env`
4. Run `php artisan migrate`
5. Run `php artisan serve`

### Notes
- Map page requires authentication
- Dummy hotel data used
